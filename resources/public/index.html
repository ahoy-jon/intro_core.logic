Functions vs Relations


Functions :

(= (f x  ) y) -> true / false
(= (g x y) z) -> true / false
(= (h    ) x) -> true / false

Relations :

(fº x y  )    -> goals
(gº x y z)    -> goals
(hº x    )    -> goals
